# itsserbin/homebrew-tap

Homebrew tap for packages maintained by **itsserbin**.

## Install

```bash
brew tap itsserbin/homebrew-tap
brew install ferrum
```

Alternative (without pre-tap):

```bash
brew install itsserbin/homebrew-tap/ferrum
```

## Upgrade

```bash
brew update
brew upgrade ferrum
```

## Uninstall
```bash
brew uninstall ferrum
```
